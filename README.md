# GridClicker

**GridClicker** 是一款跨平台的全键盘鼠标控制工具。它专为极客设计，通过在屏幕上叠加网格坐标，让用户能够仅通过键盘精准、高效地控制鼠标光标，实现点击、拖拽和滚动等操作。

目前支持 **macOS** 和 **Ubuntu (X11)** 平台。

---

## 🚀 核心通用特性

- **被动覆盖层 (Passive Overlay)**: 窗口不抢占焦点，支持点击穿透，不干扰底层应用。
- **双字母快速定位**: 将屏幕划分为 26x26 网格，通过双字母坐标（如 `AA`, `FS`）实现瞬间跳转。
- **微调模式 (Fine-Tuning)**: 使用 `HJKL` (Vim 风格) 进行像素级精准移动。
- **连续操作**: 点击、右键或移动后界面保留，支持连续操作。
- **高级动作**: 支持模拟鼠标拖拽、滚轮滚动以及一键复制并退出。

## 🎮 基础操作流 (通用)

1. **唤醒**: 轻敲各平台的 **触发键**。
2. **定位**: 输入两个字母坐标跳转并进入**微调模式**。
3. **微调与动作**:
    - `H J K L`: 移动光标。
    - `Space`: 左键点击。
    - `M`: 右键点击。
    - `T / V (Hold)`: 拖拽模式。
    - `O`: 滚动模式（使用 `J/K` 滚动）。
4. **退出**: 按 `ESC` 隐藏界面。
