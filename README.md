# GridClicker (AllInKeyboard)

**GridClicker** 是一款专为 macOS 极客打造的**全键盘鼠标控制工具**。

v1.1.0 引入了全新的 **Passive Overlay** 架构，实现了完美的“无焦点”操作，让滚动、拖拽和点击如原生般顺滑。

---

## 🚀 快速开始

### 1. 运行
```bash
# 赋予权限（如果需要）并运行
./dist/GridClicker.app/Contents/MacOS/GridClicker
```
*建议将 App 移动到 `/Applications` 文件夹以获得最佳体验。*

### 2. 核心操作流

1.  **唤醒**: 轻敲 **Left Command**。
2.  **定位**: 输入两个字母（如 `AM`）跳转光标。
3.  **操作**:
    *   **点击**: 按 `Space`。
    *   **右键**: 按 `M`。
    *   **滚动**: 按 `O` 开启滚动，然后 `J` 下滚，`K` 上滚。
    *   **拖拽复制**: 
        1. 按住 `V`。
        2. 用 `HJKL` 移动光标（此时会选中文本）。
        3. 按 `Enter` (松开 V)，程序会自动复制选中内容并隐藏。

## ❓ 常见问题 (FAQ)

### 1. 更新或移动 App 后无法唤醒？
如果你重新编译了 App 或者将它覆盖到了 `/Applications`，macOS 可能会重置其权限或将其标记为“损坏”。

**解决方法：**
1.  **清除隔离标记** (修复“已损坏”提示)：
    ```bash
    xattr -cr /Applications/GridClicker.app
    ```
2.  **重置辅助功能权限**：
    前往 `系统设置` -> `隐私与安全性` -> `辅助功能`，**删除 (按减号)** 列表中的 `GridClicker`，然后重新运行 App 并再次添加。
3.  **重启进程**：
    ```bash
    killall GridClicker
    open /Applications/GridClicker.app
    ```

### 2. 双击图标没反应？
这是正常的。GridClicker 是一个**后台守护进程 (Daemon)**，它没有 Dock 图标和主窗口。双击后它已经在后台运行，直接**轻敲左 Command 键**即可使用。
